# California Burger

Premium West Coast Burgers - Fresh, Juicy, and Delicious!

## About

California Burger is a premium burger restaurant website showcasing our signature West Coast flavors and quality ingredients.

## Technology Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
